Maps of all cages and the IJC/Book Room can be found in Spec SharePoint in [Cage layouts](https://vandalsuidaho.sharepoint.com/:f:/r/sites/Storage-Library/Documents/spec/Policies%20and%20Procedures/Cage%20layouts?csf=1&web=1&e=ZUUcwx).

To find the location of a collection, check the ArchivesSpace record for **Instances** information. Be aware that some records might have a permanent and temporary location attached. See [Temporary Locations]({{ '/content/tips-tricks/temp-locations.html' | relative_url }}) for more details. 

**Please note:** Not all records have instances attached, so if something doesn't have a clear location, see the following information for hints. Spec has long-term plans to arrange materials on shelves in a logical order and record instance for all materials.
